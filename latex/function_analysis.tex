	\section{Functions}
	
	Functions are converted in tree form for manipulation and evaluation, consider the function $f$,
	$$
		f(x, y) = (4x^y - 2\sinh y)(9xy + \cos^2 \ln x)
	$$
	the equivalent function tree is,
	
	\Tree [.$f(x,y)$ [.$\times$ 
		[.$-$ 
			[.$\times$ 
				[.4 ] 
				[.pow [.$x$ ] [.$y$ ] ]
			] [.$\times$ 
				[.2 ]
				[.sinh [.$y$ ] ]
			]
		] [ .$+$ 
			[.$\times$ [.9 ] [.$\times$ [.$x$ ] [.$y$ ] ] ] 
			[.pow 
				[.cos [.ln [.$x$ ] ] ]
				[.2 ]
			]
		]
	] ]
